# jsng shell

- based on ptpython
- allows autocompletion in js-ng world objects
- allows autocompletion of factories instances `j.clients.github.threefoldtech .. etc`
- allows coloring for different kind of attributes (methods, properties, modules..)

## running
- `poetry run jsng`
